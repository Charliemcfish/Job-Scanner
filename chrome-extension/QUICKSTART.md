# Quick Start Guide

## Installation (2 minutes)

1. **Open Chrome Extensions Page**
   - Type `chrome://extensions/` in the address bar and press Enter
   - OR click the three dots menu â†’ More Tools â†’ Extensions

2. **Enable Developer Mode**
   - Toggle "Developer mode" ON (top-right corner)

3. **Load the Extension**
   - Click "Load unpacked" button (top-left)
   - Navigate to the `chrome-extension` folder
   - Click "Select Folder"

4. **Done!**
   - The extension icon should appear in your toolbar
   - If not visible, click the puzzle piece icon and pin it

## Using the Extension

### First Time Setup

1. Go to: `https://www.upwork.com/nx/find-work/`
2. Click the extension icon
3. Toggle "Auto-Refresh" to **ON**
4. **Important:** Click anywhere on the page when you see the green notification (required for Chrome to enable sounds)
5. That's it! You'll hear a sound when new jobs appear

### Important Notes

- **First Load**: You'll hear sounds for all current jobs (they're "new" to the extension)
- **After First Refresh**: Only genuinely new jobs will trigger sounds
- **Random Timing**: Refreshes happen every 60-70 seconds (randomized)
- **Keep Tab Open**: The tab can be inactive/minimized, but don't close it

## Customizing the Sound

1. Click the extension icon
2. Click "Choose Audio File"
3. Select any audio file (mp3, wav, etc.)
4. Done! New sound will play for notifications

## Turning It Off

1. Click the extension icon
2. Toggle "Auto-Refresh" to **OFF**

## Troubleshooting

### No sound?
- Check if tab is muted (right-click tab)
- Check browser volume
- Try uploading a different audio file

### Not refreshing?
- Make sure toggle is ON
- Make sure you're on the correct Upwork URL
- Press F12 and check Console for errors

### Extension not showing?
- Make sure you selected the `chrome-extension` folder
- Check all files are present in the folder
- Try reloading: chrome://extensions/ â†’ click refresh icon
